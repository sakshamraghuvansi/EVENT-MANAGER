# Event Manager – Community local Event Discovery Platform

## GDG Hackathon Solasta 2026

EventLocal is a full-stack community event discovery platform developed for GDG Hackathon Solasta 2026 at IIITDM Kurnool.

The platform enables users to discover local events, create their own events, manage event listings, and express interest through RSVP functionality.

---

## Problem Statement

Students and community members often miss workshops, hackathons, meetups, and cultural events due to the absence of a centralized platform.

EventLocal solves this problem by providing a single platform where:

- Users can discover upcoming events
- Organizers can publish events
- Visitors can search and filter events
- Community participation is encouraged through RSVPs

## Installation

### Clone Repository

```bash
git clone <repository-url>
cd EventLocal
```

### Install Dependencies

```bash
npm install
```

### Run Server

```bash
npm start
```

Server starts on:

```text
http://localhost:3000
```
